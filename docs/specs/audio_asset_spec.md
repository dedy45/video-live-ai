# Audio Asset Specification

> **Status**: Active
> **Date**: 2026-03-07

## Reference Audio for Voice Cloning

| Property | Requirement |
|----------|-------------|
| **Duration** | 3-10 seconds |
| **Format** | WAV (16-bit PCM) |
| **Sample rate** | 16000 Hz or 24000 Hz |
| **Channels** | Mono |
| **Content** | Clear speech, no background noise |
| **Language** | Match target language (Indonesian recommended) |
| **Path** | `assets/avatar/reference.wav` |

## Quality Checklist

- [ ] No background noise or music
- [ ] No echo or reverb
- [ ] Clear pronunciation
- [ ] Natural speaking pace
- [ ] Consistent volume level
