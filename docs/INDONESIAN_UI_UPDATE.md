## Troubleshooting

### Jika UI Masih Bahasa Inggris:
1. Clear browser cache (Ctrl+Shift+Delete)
2. Hard refresh (Ctrl+F5)
3. Rebuild frontend: `npm run build`
4. Restart server

### Jika Ada Typo atau Terjemahan Kurang Tepat:
1. Edit file panel yang relevan di `src/dashboard/frontend/src/components/panels/`
2. Rebuild: `npm run build`
3. Refresh browser

## Kontak & Support
