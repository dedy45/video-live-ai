# Realism Scorecard

> **Status**: Active
> **Date**: 2026-03-07

## Scoring Dimensions

Rate each dimension from 1 (poor) to 10 (indistinguishable from real human).

| Dimension | Description | Weight | Target | Current |
|-----------|-------------|--------|--------|---------|
| **Lip sync** | Mouth movement matches audio | 25% | 7/10 | TBD |
| **Facial motion** | Natural expressions, blinking | 20% | 6/10 | TBD |
| **Voice naturalness** | TTS sounds human | 25% | 7/10 | 6/10 |
| **Timing behavior** | Response delay, pacing | 15% | 6/10 | TBD |
| **Stream stability** | No freezes, artifacts | 15% | 8/10 | TBD |

## Weighted Score

```
Score = (lip_sync * 0.25) + (facial * 0.20) + (voice * 0.25) + (timing * 0.15) + (stability * 0.15)
```

## Review Cadence

- Score after each major change
- Track improvement over time
- Minimum acceptable score for internal demo: **5.0/10**
- Target for public stream: **7.0/10**
